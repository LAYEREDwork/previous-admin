# Projekt scannen

Scanne das Projekt und aktualisiere deinen Kontext über die Codebase.

## Anweisungen

1. Analysiere die Projektstruktur (Verzeichnisse, wichtige Dateien)
2. Identifiziere verwendete Technologien und Frameworks
3. Verstehe die Architektur (Frontend/Backend Split)
4. Prüfe kürzliche Änderungen via `git status` und `git log`
5. Fasse die wichtigsten Erkenntnisse zusammen

**WICHTIG**: Ändere keinen Code! Dies ist nur eine Analyse.

Kommuniziere die Ergebnisse auf Deutsch.
