# npm Scripts anzeigen

Zeige alle verfügbaren npm run Scripts für dieses Projekt an.

## Anweisungen

1. Lies die `package.json` Datei
2. Extrahiere alle Scripts aus dem `scripts` Objekt
3. Zeige sie in einer übersichtlichen Liste mit kurzer Beschreibung

## Format

```
npm run dev          — Frontend Development Server starten
npm run backend      — Backend Server starten
npm run build        — Production Build erstellen
npm run test         — Alle Tests ausführen
...
```
