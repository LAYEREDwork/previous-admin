# Dokumentation generieren

Generiere detaillierte Dokumentation zu einem spezifizierten Thema im Kontext des Projekts.

## Verwendung

```
/docs <topic>
/docs <topic> > dateiname.md
```

## Anweisungen

1. Falls kein Topic angegeben: Generiere umfassende Projektdokumentation
2. Falls Topic angegeben: Fokussiere auf dieses spezifische Thema
3. Inkludiere Code-Beispiele wo sinnvoll
4. Verwende Markdown-Format

## Optionale Ausgabe in Datei

Falls der Benutzer `> dateiname` angibt, schreibe die Dokumentation in diese Datei im Projektstammverzeichnis.
