# Development Setup Anleitung

Erstelle eine Schritt-für-Schritt-Anleitung zum Einrichten der Entwicklungsumgebung.

## Anweisungen

1. Beschreibe die Voraussetzungen (Node.js Version, npm, etc.)
2. Erkläre die Installation (`npm install`)
3. Zeige wie man Backend und Frontend startet
4. Liste nützliche npm Scripts auf
5. Erkläre die Projektstruktur kurz

## Optionale Ausgabe in Datei

Falls der Benutzer `> dateiname` angibt, schreibe die Anleitung als Markdown in diese Datei im Projektstammverzeichnis.

## Beispiel-Struktur

```markdown
# Development Setup

## Voraussetzungen
- Node.js 22+
- npm

## Installation
npm install

## Development Server starten
# Terminal 1: Backend
npm run backend

# Terminal 2: Frontend
npm run dev

## Nützliche Commands
- npm run test — Tests ausführen
- npm run lint — Linting
- npm run build — Production Build
```
